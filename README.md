
## ニコ生 新配信録画ツール（仮）

ニコ生のHLS新配信を録画するツールです。  
このツールを改良していただいても、もし参考になるところがあれば  
ソースを使って別のツールを作っていただいても構いません。
リファクタリング大歓迎です。
